CTE DB-SE-C RD 470/2021 100% Local Sin IA

¿Qué es CadZapatas?

CadZapatas es una aplicación de escritorio (Windows) para el diseño, cálculo y documentación de cimentaciones bajo la normativa técnica española vigente.

Diseña zapatas aisladas, losas de cimentación, pilotes y muros de contención con trazabilidad completa a norma. Sin APIs, sin IA, sin servicios online — todo cálculo se ejecuta localmente en tu máquina.

✓ Normativa: CTE DB-SE-C (geotecnia) + Código Estructural RD 470/2021 (hormigón y acero)

Funcionalidades

📐 Modelado BIM

Zapatas aisladas, corridas, losas, pilotes y muros con parámetros geométricos y materiales

🧮 Cálculo geotécnico

Hundimiento (Brinch-Hansen), asientos, deslizamiento, vuelco, empujes

🏗️ Cálculo estructural

Flexión, cortante, punzonamiento, cuantías, fisuración

📄 Documentación automática

Memoria PDF, despiece XLSX, planos DXF, mediciones BC3

🎯 Motor normativo trazable

Cada comprobación genera un CalcTrace con fórmula, entradas, resultado y veredicto

💾 Persistencia local

Formato .czap (SQLite + JSON) con histórico de operaciones

Empezar

📋 Prerrequisitos

  • Windows 10/11 (requerido para WPF)
  • .NET 8 SDK — descargar
  • Git (opcional)

⬇️ Clonar el repositorio

git clone https://github.com/adrianstoot/cadzapatas.git cd cadzapatas

🔧 Compilar

dotnet restore dotnet build -c Release

✅ Ejecutar tests

dotnet test

📊 Demo end-to-end (genera PDF + BC3)

dotnet run --project samples/SampleProject

🖥️ Ejecutar la aplicación gráfica

dotnet run --project src/CadZapatas.Desktop
⚠️ Nota: Se requiere .NET 8 SDK en tu máquina. La app es nativa Windows (WPF).

Estructura del proyecto
